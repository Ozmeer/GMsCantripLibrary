# GM's Cantrip Library для MapTool

## 📖 О проекте

Коллекция простых утилитарных макросов для VTT MapTool, оформленных в виде библиотечных токенов. Основная цель - упрощение управления игровым полем, особенно полезно при оффлайн-игре на большом ТВ.

## 🎮 Библиотечные токены

### 1. Lib:GMsCantripLibrary
Сборник инструментов для управления отображением карты у игроков.

#### Доступные макросы:

** init ** (бирюзовая кнопка)
- **Назначение:** Инициализация параметров
- **Ключевая особенность:** Задает параметры defaultZoom, width, height, rotate и другие, которые лучше не трогать
- **Требует настройки:**
  1. Выставите токен на любую карту, если он еще не выставлен, и выберите его
  2. Откройте редактор макроса `init` на панели выбранного токена
  3. Выставите необходимые вам значения параметров описанных выше
  4. Сохраните макрос и один раз выполните
  5. Переносить макрос на панель кампании не требуется
  6. При необходимости процедуру настройки (шаги 2-4) можно повторить
- **Значения масштаба:** `1.0 = 100%`, `1.5 = 150%`, `2.0 = 200%` и т.д.
- **Параметры:**
  1. defaultZoom - значение масштаба по умолчанию подходящие под размер сетки не экране игроков, нужен для Player Zoom, по умолчанию 0.4056
  2. width - ширина расширения экрана игроков, нужен для Handout, по умолчанию 3840 для 4к ТВ
  3. height - высота расширения экрана игроков, нужен для Handout, по умолчанию 2160 для 4к ТВ
  4. rotate - угол поворота изображения раздатки/арта, нужен для Handout, по умолчанию -90 (градусов), если экран для игроков находится не в стандартном положении
  5. остальные являются необходимыми вспомогательными параметрами, их менять не надо

** Center Player View ** (зеленая кнопка)
- **Назначение:** Центрирует карту игроков, как у ДМа
- **Ключевая особенность:** НЕ изменяет масштаб, сохраняя установленный размер сетки
- **Использование:**
  1. Выставите токен на любую карту, если он еще не выставлен, и выберите его
  2. Перенесите макрос `Center Player View` с токена на панель кампании
  3. Нажмите для мгновенного выравнивания вида игроков на точке с центром, как на экране мастера
  4. Можно назначить горячую клавишу
- **Настройка:** Редактирование не требуется

** Player Zoom ** (фиолетовая кнопка)
- **Назначение:** Применяет предустановленный масштаб для карты игроков, заданный в defaultZoom
- **Ключевая особенность:** изменяет масштаб карты под выставленный параметр, например, чтобы сетка на экране соответствовала реальному размеру
- **Использование:**
  1. Выставите токен на любую карту, если он еще не выставлен, и выберите его
  2. Перенесите макрос `Player Zoom` на панель кампании
  3. Нажмите чтобы масштаб карты игроков соответствовал заданному
  4. Можно назначить горячую клавишу
- **Настройка:** Редактирование не требуется

** - Zoom и + Zoom ** (серые кнопки)
- **Назначение:** Меняет масштаб карты игроков с шагом 5%, позволяет отдалить или приблизить карту
- **Требует настройки:**
  1. Выставите токен на любую карту, если он еще не выставлен, и выберите его
  2. Перенесите макросы `- Zoom` и `+ Zoom` на панель кампании
  3. Можно назначить горячую клавишу
- **Применение:** Есть проверки чтобы масштаб был в пределах от 5% до 500%
- **Настройка:** Редактирование не требуется

### 2. Lib:Door
Интерактивные двери с поддержкой VBL и MBL.

#### Как это работает:

**Шаг 1: Подготовка карты**
1. Нарисуйте стены на карте, чтобы задать VBL/MBL
2. Разместите токен `Lib:Door` из библиотеки на место каждой двери (лучше НЕ копировать токены с карты)
3. *Важно:* MapTool автоматически переименует новые токены добавив счетчик (например, `Lib:Door 1`, `Lib:Door 2`)

**Шаг 2: Инициализация двери**
1. Выберите токен двери
2. В панели макроса токена нажмите `init`
3. С карты на токен перенесутся данные VBL/MBL

**Шаг 3: Использование двери**
1. Перенесите макрос `Use Door` (желтая кнопка) на панель кампании
2. Выберите любой токен двери на карте
3. Нажмите `Use Door` для переключения (открыто/закрыто)

#### ⚠️ Замечания:

- **Видимость игрокам:** Для скрытых дверей можно убрать галочку с "Видимый для игроков"
- **Визуальные эффекты:** Чтобы визуально выделить токен используйте настройки "Ореол"/"Аура" 

### 2. Lib:Handout
Токен для отображения изображений или раздаточных материалов на экране игроков.

#### Как это работает:

**Шаг 1: Подготовка токена**
1. Выставите токен `Lib:Handout` на карту, в удобное вам место, например, на место столкновения, чтобы в дальнейшем показать как выглядит монстр или NPC
2. *Важно:* MapTool автоматически переименует новые токены добавив счетчик (например, `Lib:Handout 1`, `Lib:Handout 2`)
3. Откройте параметры изображения токена
4. Добавьте в Портрет и/или Раздаточный материал те изображения из библиотеки, которые хотите отображать на экране
5. Сохраняем и закрываем настройки токена

**Шаг 2: Инициализация раздатки**
1. Выберите токен раздатки
2. В панели макроса токена нажмите `init`
3. Токен подтянет параметры из Lib:GMsCantripLibrary или задаст им значения по умолчанию, как описано в Lib:GMsCantripLibrary

**Шаг 3: Использование раздатки**
1. Перенесите макросы `Handout` и `Portrait` (бежевые кнопки) на панель кампании
2. Выберите любой токен раздатки на карте
3. Нажмите `Handout` или `Portrait`, тогда на экране игроков отобразится изображение
4. Повторное нажатие закрывает окно с изображением

#### ⚠️ Замечания:

- **Видимость игрокам:** По умолчанию галочка "Видимый для игроков" не установлена


## ⚠️ Технические ограничения

- Макросы намеренно упрощены и могут не содержать расширенной обработки ошибок
- В первую очередь предназначены для локальной игры на общем экране
- Требуется MapTool 1.10+ со стандартными настройками
